# Analytics Dashboard → React Migration

## Objective
You are an expert React developer. I am providing you with multiple 
reference assets from an existing analytics dashboard. Your job is to 
build a complete, production-ready React application that is a pixel 
perfect 1:1 recreation of the original dashboard. The final result 
must be visually and functionally indistinguishable from the original.

## Assets Provided
1. dashboard definition file — contains all widget definitions, 
   queries, layout grid and filter configurations
2. outerHTML — scraped DOM of the live dashboard with exact labels,
   axis text, KPI values, layout structure and CSS
3. static datasets — CSV/JSON data files extracted from the source
4. screenshots — images of every dashboard state

Analyze ALL assets together before writing a single line of code.
Cross reference them to resolve any conflicts, always preferring 
the screenshot as the visual ground truth.
## Deliverables

1. Complete runnable React project
2. README.md with:
   - Setup and run instructions
   - Full list of every widget migrated
   - Notes on any synthetic data used
   - Steps to connect real data API later

## Golden Rules
- Screenshots are the ultimate source of truth for visuals
- Never use placeholder grey boxes — every widget renders real data
- Always use human readable labels from outerHTML not raw field names
- Every filter must actually work, not just look correct
- The finished dashboard must look like a live production tool
- When in doubt about any detail, refer back to the screenshots
